**Tayyab-Sec CLI** is a powerful, multipurpose cybersecurity toolkit designed for penetration testers, security researchers, and students. Developed by **Tayyab** as a Final Year Project (FYP), this tool consolidates various security modules into a single, easy-to-use terminal interface.

---

## 🚀 Features

### 🔍 Pentesting
- **Advanced Port Scanner:** Deep scan for open ports and services.
- **Vulnerability Scanner:** Automated vulnerability detection using Nmap scripts.
- **Host Discovery:** Identify all active devices on a local network.
- **IP Pinger:** Quick reachability check for target hosts.

### 🌐 OSINT (Open Source Intelligence)
- **Domain WHOIS:** Full registration details for any domain.
- **Username Tracker:** Search for usernames across 50+ social platforms.
- **IP Geolocation:** Get location, ISP, and organization data for any IP.
- **Phone Lookup:** Carrier and location identification for international numbers.

### 🛠️ Utilities & Web Security
- **Phishing Link Analyzer:** Heuristic analysis for suspicious URLs.
- **Website Cloner:** Download website source code for local analysis.
- **Cookie Auditor:** Security check for session cookies (HttpOnly/Secure flags).

## 🛠️ Installation & Usage

### Prerequisites
- **Kali Linux**, **Ubuntu**, or any Debian-based distro.
- Python 3.x

### Quick Setup
Clone the repository and run the setup script:
```bash
git clone https://github.com/yourusername/tayyab-sec-cli.git
cd tayyab-sec-cli
sudo chmod +x setup.sh
sudo ./setup.sh
```

### Running the Tool
```bash
python3 main.py
```
